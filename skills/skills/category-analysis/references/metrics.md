# 指标定义

## 周期

- **周定义**：周四至周三
- **分析日期**：周四（生成报告）
- **数据截止**：周三（T-1）
- **多周范围**：从 `start_date`（如 2026-02-25）到分析日期前一天

---

## 核心指标


| 指标  | 数据库字段                     | 说明            |
| --- | ------------------------- | ------------- |
| 销售额 | `total_sale_amount`       | 品类/门店销售总金额（元） |
| 客流  | `total_customer_count`    | 购买顾客数（门店级去重）  |
| 利润额 | `full_link_profit_amount` | 全链路利润（元）      |
| 利润率 | `full_link_profit_rate`   | 利润/销售额        |


---

## 计算指标


| 指标   | 公式                                | 说明                             |
| ---- | --------------------------------- | ------------------------------ |
| 日均销售 | 周销售总额 / 天数                        | 所有金额统一用日均                      |
| 日均目标 | 来自 `config/category_targets.json` | 按 `week_label`（如 `2026-W13`）读取 |
| 达成率  | 日均销售 / 日均目标 × 100%                | 门店整体目标 = 各品类目标之和               |
| 环比   | (本周日均 - 上周日均) / 上周日均 × 100%       | 上周 = 当前周往前推 7 天                |
| 客单价  | 日均销售 / 日均客流                       | 品类维度或门店维度                      |
| PI   | 品类日均客流 / 门店日均客流 × 100             | 衡量品类客流覆盖度                      |
| 毛利率  | 利润额 / 销售额 × 100%                  | ⚠️ 数据待修复                       |


---

## 表现判断


| 状态  | 条件                  |
| --- | ------------------- |
| ✅   | 达成率 ≥ 100% 且 环比 > 0 |
| ⚠️  | 其他                  |


---

## 润色筛选标准


| 分组  | 条件（满足任一）                         |
| --- | -------------------------------- |
| 做得好 | 达成率 ≥ 80% 或 环比 ≥ +5% 或 毛利率 ≥ 25% |
| 需关注 | 达成率 < 60% 或 环比 ≤ -5% 或 毛利率 < 10% |


---

## 数值格式

- 金额 ≥ 10,000：万元（2位小数），如 1.55万
- 金额 < 10,000：元（无小数），如 2,755元
- 百分比：1位小数，如 18.3%、+31.1%
- 所有金额用**日均**，不用周累计
