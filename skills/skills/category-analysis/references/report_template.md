## 写作红线（润色时必须遵守）

1. **结论先行**：每段第一句是判断
2. **评价带基准**：说好/差必须指出对比了什么
3. **禁止编造因果**：不猜测数据外的原因
4. **禁止空洞评价**：不用"稳健"、"良好"等无数据支撑的词
5. **全文 1500-2000 字**（不含表格）
